\frame[<+->]
{
  \begin{block} {Why contracts?}
    \begin{itemize}[<1->]
      \item Allow us to be very explicit about conditions for a function's
        runtime environment.
      \item Languages like Eiffel provide a very powerful contract
        language.
      \item Approximations are possible in Python with very little work.
    \end{itemize}
  \end{block}
}

\frame[<+->]
{
  \begin{block} {What can we reason about?}
    \begin{itemize}[<1->]
      \item There are three things we'd like to provide in Python:
        \begin {itemize}[<1->]
        \item Function Preconditions
        \item Function Postconditions
        \item Class invariants
        \end{itemize}
    \item My example only provides asserts for function parameters
    \end{itemize}
  \end{block}
}
